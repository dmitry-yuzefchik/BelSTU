TTN.BY

1. Проверка работы строки поиска
    1) Перейти на главную страницу https://www.ttn.by/.
    2) Заполнить строку поиска (Прим. пылесос);
Ожидаемый результат: если данный товар присутствует, то должен отобразиться
список товаров и подходящие категории. В случае его отсутствия должно отобразиться
сообщение "Данный товар не найден".

4. Проверка удаления товаров из корзины
    1) Перейти на страницу https://www.ttn.by/.
    2) Перейти на страницу со списком товаров (Прим. https://www.ttn.by/electronics/telephony_and_communication/smartphones).
    3) Добавить в корзину товар.
    4) Кликнуть на кнопку "Ваша корзина".
	5) Кликнуть на кнопку "Очистить".
Ожидаемый результат: должна отобразится пустая корзина.

3. Проверка валидации значения фильтра по цене
    1) Перейти на страницу https://www.ttn.by/.
    2) Перейти на страницу со списком товаров (Прим. https://www.ttn.by/electronics/telephony_and_communication/smartphones).
    3) Заполнить поле стоимости значениями: от 0 до 100;
Ожидаемый результат: от 0 до 100: должен отобразиться список товаров, цена каждого не больше 100;

4. Проверка добавления товаров в корзину
    1) Перейти на страницу https://www.ttn.by/.
    2) Перейти на страницу со списком товаров (Прим. https://www.ttn.by/electronics/telephony_and_communication/smartphones).
    3) Добавить в корзину товар.
    4) Кликнуть на кнопку "Ваша корзина";
Ожидаемый результат: должна отобразиться страница со списком выбранных товаров
и формой для ввода личных данных, посчитаться общая сумма товаров. Также на иконке
корзины должно быть отображено количество товаров.

5. Проверка удаления списка желаний + 
    1) Перейти на страницу https://www.ttn.by/.
    2) Произвести регистрацию и вход в личный кабинет.
    3) Перейти на страницу списка желаний https://www.ttn.by/shop/wishlist.
    4) Нажать на кнопку "удалить список".
Ожидаемый результат: на странице должен появиться новый список с заданным названием.

6. Проверка создания списка желаний +
    1) Перейти на страницу https://www.ttn.by/.
    2) Произвести регистрацию и вход в личный кабинет.
    3) Перейти на страницу списка желаний https://www.ttn.by/shop/wishlist.
    4) Нажать на кнопку "+ Новый список".
    5) Ввести название списка и нажать "Создать";
Ожидаемый результат: на странице должен появиться новый список с заданным названием.

7. Проверка добавление товаров в список желаний +
    1) Перейти на страницу https://www.ttn.by/.
    2) Произвести регистрацию и вход в личный кабинет.
    3) Перейти на страницу списка желаний https://www.ttn.by/shop/wishlist.
    4) Нажать на кнопку "+ Новый список".
    5) Ввести название списка и нажать "Создать".
    6) Перейти на страницу со списком товаров (Прим. https://www.ttn.by/electronics/telephony_and_communication/smartphones).
    7) Открыть товар и нажать на добавление в список.
    8) Перейти на страницу списка желаний https://www.ttn.by/shop/wishlist;
Ожидаемый результат: в списке должен отобразиться добавленный товар.

8. Проверка сравнения товаров
    1) Перейти на страницу https://www.ttn.by/.
    2) Перейти на страницу со списком товаров (Прим. https://www.ttn.by/electronics/telephony_and_communication/smartphones).
    3) Нажать на "Добавить к сравнению" на нескольких товарах.
    4) Открыть страницу сравнения товаров https://www.ttn.by/shop/product-compare/compare;
Ожидаемый результат: на странице должны отобразиться выбранные товары с подробными характеристиками,
разные значение характеристик должны быть выделены оранжевым цветом.

9. Проверка изменения имени пользователя +
    1) Перейти на страницу https://www.ttn.by/.
    2) Произвести вход в личный кабинет.
	3) Перейти на страницу личного кабинета.
	4) Имзенить поле имя
	5) Нажать кнопку сохранить
Ожидаемый результат: страница должна обновится, имя должно измениться.

10. Проверка входа в личный кабинет пользователя +
    1) Произвести регистрацию на странице https://www.ttn.by/login.
    2) Выбрать пункт "Войти" и ввести данные указанные на прошлом шаге.
Ожидаемый результат: в случае успешного входа должно произойти перенаправление
на страницу личного кабинета пользователя со список личных данных.
